# 🍽️ Student Meal Nutrition Calculator Web App

## 📌 Overview

The **Student Meal Nutrition Calculator** is a web-based application designed to help students analyze the nutritional value of their daily meals. Users can input any food item, and the app calculates key nutritional components like **calories, protein, carbohydrates, and fats**.

This project aims to promote **healthy eating habits** and provide **quick nutritional insights** in a simple and interactive way.

---

## 🚀 Features

* 🥗 Add **any custom meal or food item**
* 🔍 Fetch real-time nutrition data using API
* 📊 Displays:

  * Calories
  * Protein
  * Carbohydrates
  * Fats
* ⚡ Fast and responsive UI
* 🎨 Clean design with animations
* 📱 User-friendly interface

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/your-username/student-meal-nutrition.git
cd student-meal-nutrition
```

### 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

### 3️⃣ Add API Key

Open `app.py` and replace:

```python
API_KEY = "your_api_key_here"
```

### 4️⃣ Run the App

```bash
python app.py
```

### 5️⃣ Open in Browser

```
http://127.0.0.1:5000/
```

---

## 💡 How It Works

1. User enters a meal (e.g., "2 roti and dal")
2. Backend sends request to nutrition API
3. API returns nutritional data
4. Results are displayed on UI instantly
